# Obsidian_excalidraw_addon_test

В планах навайбкодить аддон для графического редактора Obsidian.  
Аддон исправит некоторые недоработки Obsidian и его плагина Excalidraw.  

Примеры таких недоработок или багов в Obsidian:  
- При alt + tab (и при переключении между вкладками) пропадает выделение с текста в excalidraw. Это неудобно: приходится выделять его снова, чтобы писать дальше.  
- В каких-то случаях с интерфейса Excalidraw пропадает фокус - не работают hotkeys для выбора инструментов - приходится кликать на него, чтобы переключить фокус.
