# Obsidian_excalidraw_addon_test

В планах навайбкодить аддон, который исправит некоторые недоработки Obsidian и Excalidraw.  

Примеры таких недоработок или багов:  
- При alt + tab пропадает выделение с текста, неудобно.  
- В каких-то случаях с интерфейса excalidraw пропадает фокус - не работают hotkeys для выбора инструментов.
